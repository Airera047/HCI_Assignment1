Background Overview

What is Blackboard?

Blackboard is a web-based course-management system designed to facilitate student’s learning experience through access to multiple online materials and activities provided by instructors and to enhance communication between both student-student and student-instructor interaction . Some basic built-in function includes discussion boards, online quizzes, notification reminder, and etc. The Blackboard is utilized in the majority of JHU courses for course materials sharing, assignment submission/grading and interaction between Students, TA and Professors. Contextual Inquiry

In order to know better how the blackboard can be improved, a contextual inquiry method is applied to examine some spotted problems under regular circumstances.

The following tasks includes problems I’ve met during semesters with Blackboard and by dividing them separately into three task for two users, we can see if and where they met problems and make improvement accordingly.

Task 3

In order to meet the user's need of interpreting the grade, I add a grade interpretor in my grade section that would help student a lot in making important academic decision

Task 2

For solving the Assignment download submission and due date problem, right now, all the assignment is shown on the calendar and student can simply click into each assignment on their calendar and specific information shows up

Task 1

For solving the the lecture zoom link or office hour link missing problems, I design an agenda page allows all links and course information posted here so that student can easily get access to any course immediately

Since every course related items is on the calendar, the interface allows students to change their calendar setting with high flexibility and if they drop from one course they can immediately remove this course's calendar information out.

Such calendar based system can largely increase student's time sensitivity and largely prevent missing homework or event and provide a clearer regulation system for students to structure their time

Some Positive take away

The user likes the calendar based system more than previous system

Assignment download, info, submission becomes pretty clear

Accumulative Grade window can help understand grade better

Agenda list all the thing need to be done today and facilitate time management

Those Lecture Zoom Link and OH zoom link is easily accessed by student in one position

Some Place need Improvement

Contains zoom link and assignment upload/submission in agenda and calendar may not be intuitive to student

User hesitate in finding the zoom link and assignment in agenda

Not Friendly to those who hate using calendar

May cause pressure if user see an extremely compact schedule
